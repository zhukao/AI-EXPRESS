Workflow¶

  1. Clone

  2. Install code style tools

pip install pre-commit==1.17.0
apt install clang-format-9
ln -s /usr/bin/clang-format-9 /usr/bin/clang-format

  1. At your local repo root, run

pre-commit install

  1. Make change to your code and test. You can run all the existing unittests by the following command:

# TODO
unittest ...

Then commit your change to the local branch using git commit

  1. Make pull request:

arc diff

  1. Change your code based on review comments. The new change should be added as NEW commit to your previous commits. Then you should update the revision

arc diff --update

  1. Publish an accepted revision after review

arc land
